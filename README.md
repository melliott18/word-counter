# Word Counter

<p>C program that reads in all of the words in the files specified and prints each word found along with the number of occurrences of that word.</p>
<p>Author: Mitchell Elliott</p>
<p>Email: mitch.elliott@pacbell.net</p>

```
make
make all: Build and compile everything
make clean: Remove object files
make format: Run .clang-format
```

### Run:

```
usage: wordcounter [-s] [-v] -i <input file> -o <output file>
```
